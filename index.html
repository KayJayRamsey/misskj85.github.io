Summary: Latitude vs. X

This project was created to visualize and analyze the weather of multiple cities around the world based on their distance from the Equator.

To perform this analysis, we had to pull data from the OpenWeatherMap API and created a dataset that included over 500 cities.

After creating the dataset, we used Matplotlib to plot the results. These results included various comparisons between weather and latitude. This site displays the visualizations that were created as well as the data.
